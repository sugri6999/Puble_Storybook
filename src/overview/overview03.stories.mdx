## API와 에러 처리

<br />

> #### API 호출 시 에러 처리를 항상 포함해야 한다.

```javascript
const fetchData = async () => {
    try {
        const response = await fetch('/api/data');
        const data = await response.json();
        setData(data);
    } catch (error) {
        console.error('Error fetching data:', error);
        setHasError(true);
    }
};
```
